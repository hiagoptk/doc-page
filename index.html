Gradiente Linear

Gradientes são uma forma de conseguirmos um efeito suave de transição entre 2 ou mais cores. Temos dois tipos de gradientes no CSS, e um deles é o Linear.

Esse gradiente tipo de gradiente pode ter diversas direções, de cima para baixo, da esquerda para a direita, ou o oposto. Pode também ser na diagonal de qualquer um dos cantos

Sintaxe

Para conseguir um gradiente é necessário definirmos ao menos dois pontos de cores, podendo também definir a direção tomada pelo gradiente. Veja:

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

Default

Por padrão, a direção do gradiente linear é semre de baixo para cima, então se não definirmos a direção, e settarmos apenas os pontos de cor, ele resultará em algo mais ou menos assim:

#div { background-image: linear-gradient(red, yellow); }

Linear Horizontal

Para a construção de um gradiente horizontal, basta definirmos para qual direção nosso gradiente vai apontar, seja pra esquerda seja pra direita, usamos to left ou to right pra isso.

#div { background-image: linear-gradient(to right, red , yellow); }

Linear diagonal

Já para a criação de um gradiente podemos usar direções de forma semelhante à to right/to left, ou usarmos graus para isso

#grad { background-image: linear-gradient(to bottom right, red, yellow); }
#grad { background-image: linear-gradient(-90deg, red, yellow); }

Usando multiplas cores

Conseguimos usar mais de duas cores num gradiente linear, bastando pra isso acrescentarmos mais cores a linha de código do css. Veja:

#grad { background-image: linear-gradient(red, yellow, green); }
#grad { background-image: linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet); }

Usando transparência

O gradiente no CSS também suporta o uso de transparência, que pode ser usada pra criar os famosos Fade Effects. Nesse caso, a sintaxe muda um pouco e novos conceitos precisam ser reavaliados.

O a transparencia no CSS é definida entre 0 (transparente) e 1 (opaco). Por exemplos, o código rgb para vermelho é RGB(255, 0, 0) onde cada casa separada pela virgula corresponde a uma das 3 cores de RGB(Red = 255, Green = 0, Blue = 0). A transparência é definida acrescentando mais uma casa aos parênteses, ficando algo como RGB(255, 0, 0, 0.5). A casa da transparência pode ser ocupada por números flutuantes, como no exemplo dado, onde a transparência do vermelho ficou em 50%.

O exemplo abaixo retorna o seguinte resultado:

#grad { background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1)); }

Repetindo um gradiente linear

O gradiente no css tem uma function chama repeating-linear-gradient() usada exatamente pra isso. Veja:

#grad { background-image: repeating-linear-gradient(red, yellow 10%, green 20%); }
